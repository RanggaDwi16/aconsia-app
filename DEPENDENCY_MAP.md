### Pattern 2: Current User Helper


### Pattern 3: Auth State Stream

```dart

  data: (user) => user != null ? HomePage() : LoginPage(),
  loading: () => SplashScreen(),
  error: (e, s) => ErrorPage(),
);
```

### Pattern 4: Complex State Management

```dart
// Create StateNotifier

// Use in UI
final state = ref.watch(myStateProvider);
```
